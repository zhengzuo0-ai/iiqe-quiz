## 一、官方研习手册关键考点提取

### 卷三（长期保险）核心考点

#### 第1章：长期保险简介与产品类型
| 考点 | 关键细节 |
|------|----------|
| 人寿保险基本原则 | 最高诚信、可保利益、近因、弥偿、分摊、代位追偿 |
| 可保利益（Insurable Interest）| 源自《保险条例》第64B条；自己和配偶=无限额推定；第64A条=父母/监护人对未满18岁未成年人 |
| 可保利益时间要求 | 人寿保险：仅投保时需要；海上保险：仅损失时需要 |
| 最高诚信 | 重要事实（material fact）= 影响审慎保险人决定的任何情况；包括不披露和失实陈述 |
| 不可争议条款 | 保单生效指定期限后（通常2年），保险人不得以非欺诈性不披露争议保单 |
| 定期寿险（Term） | 纯保障无储蓄；递减定期（按揭/信贷）、可续保、可转换 |
| 储蓄寿险（Endowment）| 期满或身故均赔付面额；结合定期险+纯储蓄 |
| 终身寿险（Whole Life）| 终身保障；均衡保费制；有现金价值 |
| 万能寿险（Universal Life）| 弹性保费、可调身故保障、分拆定价（纯保障+费用+现金价值）|
| 投连险（ILAS/Unit-Linked）| 保单价值与投资基金单位挂钩；价值可升可跌 |
| 即期年金 | 一次缴费后立即开始支付；HKMC年金计划：65岁以上永久居民，保证期105%保费 |
| 延期年金 | 积累期+年金期；QDAP税扣上限$60,000/人/年（夫妇合共$120,000）；最高税率17%→最高节省$10,200 |
| 团体保险 | 供款式(contributory)与非供款式(non-contributory)；积极工作条款 |
| 均衡保费制 vs 自然保费制 | 自然保费制=每年按风险重新计算→逆选择→已废弃；均衡保费制=固定年缴保费→产生现金价值 |
| MPF强积金 | 2000年12月实施；雇主雇员各5%供款 |

#### 第2章：保单条款与条件（附加保障及保单条文）
| 考点 | 关键细节 |
|------|----------|
| 豁免保费附加保障（WP Rider）| 全残时豁免续期保费，保单继续有效 |
| 伤残定义 | 无法执行本职工作的基本行为；或双目失明/双手双脚丧失使用能力 |
| 转让（Assignment）| 绝对转让 vs 有条件转让；人寿保单可无需保险人同意转让 |
| 受益人指定 | 可撤销 vs 不可撤销受益人 |
| 保单贷款（Policy Loan）| 以现金价值为抵押借款 |
| 不没收权益（Nonforfeiture）| 现金价值可用于：退保金、缴清保险（减额缴清）、自动垫缴保费 |
| 复效（Reinstatement）| 需提供可保性证明；通常在失效后一定期限内申请 |

#### 第3章：核保与保费
| 考点 | 关键细节 |
|------|----------|
| 保费三要素 | 死亡率（mortality）+ 利息（interest）+ 费用（expenses）|
| 净保费 = 死亡率 + 利息 | 又称纯保费（pure premium）|
| 毛保费 = 净保费 + 附加费用 | loading包括营运成本、佣金、税项、灾难准备金 |
| 分红保单（PAR）vs 非分红（NON-PAR）| PAR保费较高但参与分红；定期险通常为NON-PAR |
| 死亡表（Mortality Tables）| 预测死亡率的统计基础 |
| 核保因素 | 年龄、性别、健康、职业、生活习惯、家族病史 |
| 年金核保 | 与人寿保险相反：年龄越大→年金金额越高；男性>女性（同龄）|

#### 第4章：理赔与赔付
| 考点 | 关键细节 |
|------|----------|
| 理赔所需文件 | 死亡证明、保单正本、索赔表、受益人身份证明 |
| 保险投诉局（ICB）| 前身为ICCB；投诉委员会可超越严格的合约解释 |
| 近因在人寿的应用 | 较少使用，因排除条款少；自杀排除仅限定期间 |
| 弥偿不适用人寿 | 人寿保险为利益保单（benefit policy）非弥偿保单 |

#### 第5章：行业法规与合规
| 考点 | 关键细节 |
|------|----------|
| 冷静期（Cooling-off Period）| GL25/GL26相关指引 |
| 利益说明（Benefit Illustration）| GL28 |
| 保单替换（Policy Replacement）| GL27 |

### 卷一（保险原理与实务）核心考点

#### 第1章：风险与保险
- 纯粹风险 vs 投机风险（只有纯粹风险可保）
- 特定风险 vs 基本风险（基本风险通常不可保）
- 风险管理工具：回避、预防、减少、转移、融资
- 保险的首要功能（风险转移）和次要功能（就业、储蓄、经济发展等）

#### 第2章：合约法与代理法
- 合约六要素：要约、承诺、对价、缔约能力、合法性、创设法律关系的意图
- 无效(void)/可撤销(voidable)/不可强制执行(unenforceable)合约
- 代理关系：明示/默示/表见/紧急/禁反言授权
- 代理人职责：服从、亲自执行、应有技能、忠诚、问责
- 第三人权利条例（Cap.623）

#### 第3章：可保利益
- 财产保险：损失时需要可保利益
- 人寿保险：仅投保时需要
- 海上保险：仅损失时需要
- 转让：保单转让 vs 保险金转让

#### 第4章：最高诚信
- 重要事实的法定定义
- 不需要披露的事实（常识、保险人已知/应知的事实）
- 披露义务期间：投保时（普通法）、续保时（非寿险）、保单变更时
- 违反后果：可在合理时间内从保单成立时撤销合约

#### 第5-7章：近因、弥偿及其推论、保险市场与监管
- 近因规则应用（链式事件、并发原因）
- 弥偿方法：现金、修复、替换、恢复原状
- 分摊（contribution）五要素
- 代位追偿（subrogation）四种来源
- 不足额保险与比例条款（average clause）
- 保额免赔（excess/deductible）vs 免赔额条款（franchise）
